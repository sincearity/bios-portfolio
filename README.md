# bios-portfolio

Modern portfolio built with **Next.js (App Router)**, **TypeScript**, and **Tailwind CSS**.

## Scripts
- `npm run dev` – start local dev
- `npm run build` – production build
- `npm run start` – run production server
- `npm run lint` – lint code
